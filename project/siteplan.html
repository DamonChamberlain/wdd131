site purpose üß≠

this site is a go-to spot for new devs learning the basics (html, css, js). it will share **simple tutorials, quick cheat sheets, and project ideas** to help students practice what they learn. it's a useful extra learning tool.

who's the site for? students/beginners. here are some things they might ask:

  1. **"i need a simple example for laying out my site header with flexbox. got any code?"** (for the css page)
  2. **"how do i use local storage the right way for small stuff like saving user settings?"** (for the javascript page)
  3. **"where's the quick guide for making a basic, accessible html form?"** (for the html cheat sheet)
